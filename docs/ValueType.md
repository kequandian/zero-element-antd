### ValueType
## # Tag
1. Tag所定义的属性
    1. options
        这是默认配置项,它拥有两个属性，分别是map,chy
        1. map
            如果有多个值，map属性可以为各个值设立名称
        2. chy
            chy属性可以为每个值设立state
    2. data
        1. text
            text属性可以设置标签文本，默认为"-"
    3. theme
        theme属性可以设置全部风格
    4. type
        type属性可以设置全部类型
    5. state
        state属性可以设置全部状态
